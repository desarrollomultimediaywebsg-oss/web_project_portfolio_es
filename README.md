# Portafolio Web — Sthefany Gaitán

## Descripción del proyecto

Este proyecto es un portafolio web responsivo desarrollado como parte del Sprint 4. Su objetivo es presentar mi perfil profesional como desarrolladora web y diseñadora UX, mostrando mis habilidades, herramientas y proyectos destacados mediante una interfaz moderna, accesible y adaptable a diferentes tamaños de pantalla.

El sitio incluye:

- Sección hero de presentación.
- Sección de habilidades y herramientas.
- Tarjetas de proyectos destacados.
- Footer de contacto con redes sociales.
- Diseño responsivo para desktop, tablet y mobile.
- Efectos visuales modernos utilizando gradientes y hover effects.

## Tecnologías y técnicas utilizadas

### Tecnologías

- HTML5
- CSS3

### Técnicas implementadas

- Metodología BEM
- Flexbox

- Gradientes y efectos visuales
- Semántica HTML
- Organización modular de archivos
